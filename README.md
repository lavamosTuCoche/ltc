# LavamosTuCoche - TFG 2026
Sistema de gestión para lavadero de coches desarrollado con arquitectura **MVC** y **PHP**.

## Tecnologías utilizadas
* **Backend:** PHP 8.3 + Composer
* **Servidor:** Nginx en Ubuntu Server 24.04
* **Despliegue:** CI/CD con GitHub Actions (Self-hosted runner)

## Instalación en local
1. Clonar el repo.
2. Ejecutar `composer install`.
3. Configurar el archivo `.env` con las credenciales de BD.
